# 🎬 Projeto Vídeos

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)
![YouTube](https://img.shields.io/badge/YouTube-FF0000?style=for-the-badge&logo=youtube&logoColor=white)

Esse é um projeto simples de front-end feito para praticar HTML e CSS. A ideia foi criar uma página inicial com alguns vídeos legais para assistir, usando miniaturas clicáveis que levam para outras páginas do projeto.

## ✨ Sobre o projeto

O projeto funciona como uma pequena galeria de vídeos. Na página principal, aparecem algumas opções com temas variados.

Ao clicar em uma miniatura, o usuário é levado para uma página própria, que pode conter um vídeo incorporado do YouTube, um link externo relacionado e um botão para voltar ao início.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- `iframe` para incorporar vídeos
- Links internos e externos
- Imagens como miniaturas e botão de navegação

## 📚 O que eu pratiquei

- Criação de várias páginas HTML conectadas entre si
- Organização de arquivos em pastas
- Uso de imagens clicáveis
- Inserção de vídeos externos do YouTube
- Navegação entre páginas do mesmo projeto
- Estilização básica com CSS

## 💡 Tipo de projeto

Esse é um projeto de estudo, criado durante meus primeiros passos com desenvolvimento web. Mesmo sendo simples, ele ajudou a entender melhor como páginas HTML podem se conectar e formar um pequeno site navegável.
